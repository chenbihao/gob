中间件命令

命令介绍：

middleware 生成

前置需求：

app

支持命令：

./gob middleware  			// 打印帮助信息
./gob middleware list  		// 列出所有的业务中间件
./gob middleware new  		// 创建一个新的业务中间件
./gob middleware migrate 	// 迁移 Gin 已有的中间件

支持配置：无

使用方法：

稍后补全，可以使用gob [command] help命令获取相关帮助

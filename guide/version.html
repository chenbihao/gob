版本

命令

相关的命令详见：version

gob 提供了查询当前版本和获取最新版本日志的命令

查询当前版本

使用命令 gob version

> ./gob version
gob version: 1.0.0

获取最新的版本

使用命令 gob version list

> ./gob version list

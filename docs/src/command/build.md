# 编译命令

## 命令介绍：
提供快捷 build 命令
## 前置需求：无
## 支持命令：
```sh
./gob build self		// 自编译，编译 gob 命令
./gob build backend		// 使用 go 编译后端
./gob build frontend	// 使用 npm 编译前端
./gob build all			// 同时编译前端和后端
```
## 支持配置：无
## 支持环境变量：无

## 使用方法：
稍后补全，可以使用`gob [command] help`命令获取相关帮助
